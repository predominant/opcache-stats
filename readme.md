# OpCache Statistics #

This is a basic Zend OpCache statistics page, based heavily on the stats page that is shipped with APC.

Simply host the `opcache.php` file somewhere on your webserver, and access it to view the statistics.

# Requirements #

* Zend OpCache (Duh...)
* GD
